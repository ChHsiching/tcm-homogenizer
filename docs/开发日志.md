# 中药配比客户端开发日志

## 项目概述
本项目是一个基于Web技术的中药配比客户端软件，采用前后端分离架构，支持用户管理、中药配比计算等功能。

## 技术栈
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **后端**: Python Flask
- **客户端**: Electron
- **测试**: Playwright + Puppeteer
